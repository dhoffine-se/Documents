let and const exercise 1/23/22




What is the difference between var and let?
* Both are variable declarations, but var is function blocked, and let is block scoped.




What is the difference between var and const?
* Var is function blocked and can be changed, while const is blocked scoped and also cannot be updated or re-declared.


What is the difference between let and const?
* While both are block-scoped, let can be updated but const cannot.


What is hoisting?
* Hoisting is when all declarations are moved to the top of the scope before code execution.
